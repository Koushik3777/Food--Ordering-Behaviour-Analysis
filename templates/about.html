Gastronomy Analytics Guide

A comprehensive walkthrough of the integration architecture, dashboard features, and instructions on extracting value from the visual insights.

Analytics Dashboards

Our dashboards are segmented into distinct operational sectors to assist decision-makers:

  • Cuisine Demand Analysis: Discover which food types dominate volume, total pricing share, and rating quality.
  • Ordering Time Patterns: Track order distributions by day of the week to allocate delivery staff effectively.
  • Operational Speed: Cross-reference delivery times with restaurant parameters to optimize logistics.
  • Pricing Trends: Analyze the cost of orders versus customer satisfaction scores.

Interactivity Guide

Tableau visualizations are fully dynamic. Here is how to make the most of them:

  • Hover for Details: Hover over individual bars, map coordinates, or line points to view precise numbers.
  • Dynamic Filtering: Use the filtering menus (dropdowns, checkmarks) in the dashboard sidebars to focus on specific cities, restaurants, or cuisines.
  • Multi-Select: Hold Ctrl (or Cmd on Mac) and click on multiple bars or points to slice data simultaneously.
  • Reset Controls: Use our custom "Reset" button in the app header to instantly wipe all custom filters.

Technical Implementation Stack

This website brings together a lightweight Python web framework and modern client-side embedding techniques to optimize rendering and loading performance:

Flask & Jinja2

Handles backend server routing, page templating, and serving of static assets safely and reliably.

Tableau Embedding API v3

Leverages Tableau's native JavaScript web components (<tableau-viz>) instead of static iFrames, enabling custom event handling, resize responsiveness, and programmatic filtering.

Vanilla Modern CSS

Designed using modern CSS properties (Custom Variables, Flexbox, Grid, CSS transitions, and high-performance WebKit backdrop blurs) to create an interface that feels responsive and premium.
