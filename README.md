# 🍽️ Food Ordering Behaviour and Consumer Trends: A Structured Analysis of Choices and Habits

## 📌 Project Overview

The food delivery industry is becoming increasingly competitive, with businesses facing challenges in understanding customer behaviour, optimizing delivery performance, and maximizing revenue. Despite having large volumes of data, it is often difficult to identify the key factors that influence order demand, customer spending patterns, and service efficiency.

This project addresses that gap by analysing food ordering data using **Tableau** to uncover meaningful patterns and trends, then delivering the analysis through a responsive **Flask** web application. The dataset includes key attributes such as **Order ID, Order Value, Delivery Fee, Cuisine, Meal Type, City,** and **Time Taken for Delivery**. Through interactive visualizations, the project enables a deeper understanding of how these factors impact order volume, customer preferences, and operational performance — surfacing insights such as top-performing cities, high-demand cuisines, peak mealtimes, and delivery time distribution.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Food-Ordering-Behaviour-and-Consumer-Trends.git
```

### Navigate to the project folder

```bash
cd Food-Ordering-Behaviour-and-Consumer-Trends
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
